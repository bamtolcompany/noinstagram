컴퓨터 사용법

1. 컴퓨터 키는법: 전원 버튼을 누르면 컴퓨터가 켜집니다.
2. 마우스 사용법: 마우스의 왼쪽을 누르면 무언가를 클릭할수 있습니다. 오른쪽을 누르면 메뉴가 나타납니다. 마우스 휠을 이용해서 위아래로 이동할수 있습니다.
3.키보드 사용법: 키보드를 누르면 글자를 입력할수 있습니다. ctrl + c를 누르면 무언가를 복사할수 있고, ctrl + v를 누르면 붙여넣을수 있습니다.
4. 브라우저 열기: 윈도우 컴퓨터에는 엣지, 맥os에는 사파리가 기본 브라우저로 설치되어있습니다. 아이콘을 눌러 들어가면 상단에 주소창이 나옵니다.
주소창에 키보드를 이용해서 검색어를 입력하거나 가고싶은 사이트의 주소를 입력합니다.
5.파일 열기: 파일 탐색기를 열어 원하는 파일을 두번 눌러 파일을 열수 있습니다. 저장할때는 저장을 누르거나 ctrl + s를 누릅니다.
6.컴퓨터 종료하기: 컴퓨터 하단에 시작 버튼을 누른 후, 전원 버튼을 누르고 시스템 종료를 누릅니다. 전원을 끄기 전에는 프로그램을 모두 닫고 끄는것이 좋습니다.
